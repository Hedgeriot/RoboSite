Further Learning

Codeacademy is a wonderful resource that teaches you code at your own pace! While it doesn't cover C, it teaches many basic programming fundamentals.


Learn C the Hard Way is a rigorous, hands-on tutorial on learning C. Not everything they cover will be used in Robotics, but it's a great way to get into the monster that is C.
